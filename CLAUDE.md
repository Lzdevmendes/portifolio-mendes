# Portfolio Mendes

Next.js 16 portfolio with static export deployed to GitHub Pages via GitHub Actions.

## Commit style

Always single-line commits — title only, no body, no bullet points, no co-author signatures.

Format: `type(scope): short description`

Examples:
- `feat(hero): add open to work badge`
- `fix(navbar): correct mobile menu spacing`
- `style(about): adjust card padding`

Never add a body, bullet points, or any footer to commits.
